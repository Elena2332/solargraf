Descomprimir y corregir código CSS y JavaScript



Descompresor para extraer y expandir archivos comprimidos y minimizados usando cualquier método de compresión y restaurarlos con su formato predeterminado. Revisar y corregir hojas de estilo y scripts, aunque no estén comprimidos, usando JS Beautifier.

Con la herramienta que compartimos en esta página es posible restaurar los archivos de estilo CSS y scripts, que han sido comprimidos usando cualquier servicio, como el Compresor CSS de nuestro sitio o cualquier otro disponible online.
Además de eso el código resultante es completamente limpio, usando la sintaxis y estructura correcta para cada formato.
Es de mucha utilidad para hacer legibles, comprender el código y editar cualquiera de estos archivos minimizados.
También para re-estructurar y ordenar correctamente el código en cualquiera de estos formatos, incluyendo los que están en formato JSON.



Descomprimir CSS y JavaScript

Introduce o pega el contenido del código a descomprimir en el siguiente cuadro de texto y presiona el botón correspondiente.
En el cuadro inferior se puede comprobar el resultado, correctamente formateado.



Resultado
function fcm(s) {
    var i = s.search(" ");
    var c = s.substr(0, i);
    var m = s.slice(i + 1);
    var t = c[0];
    if (t == 's') document.getElementById(c).innerHTML = m;
    else if (t == 'v') document.getElementById(c).value = m;
    else if (t == 'c') document.getElementById(c).checked = Number(m);
    else if (t == 'i') {
        if (m == "0") document.getElementById(c).style.display = 'none';
        else document.getElementById(c).src = m
    } else if (t == 'g') {
        var v = Number(m);
        if (c[1] == '1') vg('g1', v, 0, 99, '#3399ff', '%', '');
        else if (c[1] == 'p') vpot = v;
        else if (c[1] == 't') vtf = v;
        else if (c[1] == '2') {
            vta = v;
            vg('g2', 0, 0, 0, '#ffcc66', '', 'min')
        }
    } else if (t == 'p')
        if (m[0] == '1') fis('ip', 'play.png');
        else fis('ip', 'pau.png')
}

Restaurar scripts y CSS minimizados

Los archivos de estilo CSS y el código JavaScript, generalmente se comprimen y minimizan para usarlos y distribuirlos en internet.
En nuestro sitio compartimos dos herramientas para estos propósitos: Comprimir código CSS y Comprimir código JavaScript.
Al optimizarlos para la web se reduce el tamaño de los archivos y se agiliza la carga de las páginas que los usan.
Al compactarlos se eliminan espacios entre caracteres, lo que hace difícil su posterior edición.
Para poder hacer cambios en el código cuando sea necesario, debemos descomprimirlos y restaurarlos a su formato predeterminado.
Esa tarea los realizan los scripts que usa la herramienta que compartimos en esta página.
Su principal virtud es la de revisar y corregir el código, devolviendo el resultado con la sintaxis correcta en cada caso.
Además se pueden emplear para poder hacer legibles los archivos en formato JSON (formato para el intercambio de datos).



Revisar y corregir el formato del CSS y JavaScript

La herramienta también se puede emplear para revisar y corregir cualquier código en estos formatos, aunque no estén comprimidos.
Por ejemplo al introducir la siguiente regla en CSS:
.clasex{
font-size : 24px;
color:#0d47a1;  }
Obtendremos como resultado:
.clasex {
    font-size: 24px;
    color: #0d47a1;
}
Se puede comprobar su correcto indentado y el ordenado lógico de los bloques de código, para hacerlo más legible.
Un resultado similar se obtiene con el código en JavaScript.
